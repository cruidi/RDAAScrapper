# RDAAScrapper
Air Asia Invoice Scrapper
go to the file and insert your Airasia username and email
AIRASIA_EMAIL    = "username"
AIRASIA_PASSWORD = "password"

This tool will auto login based on the User name and password, and login

a 2FA will pop where you need to insert the 2FA from your email

Version 1 of the tool will only download all past the Invoice/receipt in 2026

modify the years as you wish at line 138 

pre-requisite
Requirements:
    py -m pip install playwright
    py -m playwright install chromiu
